# Notes App

A React + TypeScript notes application (Evernote replacement) with localStorage persistence.

## Features

- **Three-panel layout**: Sidebar | Note List | Note Editor
- **Full CRUD**: Create, edit, delete notes, notebooks, and tags
- **Pin & Archive**: Organize notes with pin/unpin and archive/unarchive
- **Real-time Search**: Filter notes by title or content
- **Tag Support**: Assign multiple tags to notes
- **Auto-save**: Changes save automatically as you type

## Tech Stack

- React 18 + TypeScript
- Vite
- TailwindCSS
- Zustand (state management)
- Vitest (testing)

## Getting Started

```bash
cd notes
npm install
npm run dev
```

## Build

```bash
npm run build
npm test
```
